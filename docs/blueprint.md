# **App Name**: ClickVid Pro

## Core Features:

- Secure User Roles: Secure user authentication and role-based access control (Admin, User) with IP-based restrictions to prevent account sharing and unauthorized access. Protection against API abuse, SQL injections, webapp cracks, brute-force attacks, and cybersecurity threats.
- AI Video Editing: AI Video Editor with features like adding captions, B-rolls, autozoom, transitions, and a Viral Shorts Maker. AI-powered script generation, voice cloning, transcription, voice-over, background removal & change, avatar generation.
- AI Content Generation: AI-powered tool to generate multiple types of creative content, which will let you create videos from images with music and effects, and extract media from webpages via URLs.
- Cloud Integration: Integrate Wasabi and Bunny.net for secure private file storage and CDN using pre-signed URLs; implement payment provider integrations (Stripe, Paystack, PayPal, Flutterwave).
- Video Sharing: Enable video CTA placement with integration with social media platforms (YouTube, Instagram, TikTok, Snapchat).
- AI Model Switching: Ability to dynamically switch AI models, as a tool to ensure continuous functionality, for use in content creation and chatbots.
- Dashboard Analytics: Provide dashboard notifications, app usage analytics, user activity, and reports for both admin and user roles.

## Style Guidelines:

- Primary Blue: #3B82F6 (Royal Blue) - Strong, reliable, and tech-oriented.
- Secondary Blue: #1E3A8A (Dark Navy) - For backgrounds and deeper elements, adding sophistication.
- Accent/Highlight: #FBBF24 (Amber Yellow) - A warm, inviting, and attention-grabbing contrast.
- Dark Background: #1A202C (Dark Slate)
- Light Background/Cards: #F3F4F6 (Light Gray) - For sections that need more visual separation.
- Text: #FFFFFF (White) or #1F2937 (Dark Gray) depending on background.
- Body and headline font: 'Nunito' sans-serif for a friendly, modern and easily-read look. Note: currently only Google Fonts are supported.
- Use a set of consistent, minimalist icons for all features.
- Maintain a consistent grid-based layout for intuitive navigation.
- Use subtle animations for transitions and interactions.
